Pseudo-elementos

Usado para adicionar elementos extras na pagina, sem a necessidade de declarã-lo no html, ou para selecionar alguma parte de algum elemento.

Identificado por :: palavra chave

Pseudo-elementos - ::first-letter

Selecina a primeira letra da primeira linha

Propriedades disponiveis:

  • color
  • background
  • margin
  • padding
  • border
  • font
  • Propriedade de texto

Pseudo-elementos - ::first-line

Seleciona a primeira linha de um texto

Propriedades disponivel

  • color
  • backgound
  • font
  • propriedade de texto

Pseudo-elemento - ::backdrop

Esteliza o plano de fundo de um elemento em modo de tela cheia como dialog ou video

Pseudo-elemento - ::marker

Permite estilizar os topicos ou numeros de uma lista ou entao a flexa de um elemento summary

Propriedades aceitas:
  • color
  • content
  • white-space
  • font
  • animation
  • transition

Um place-holder é uma dica para ajudar no preenchimento de algum campo de formulario.
